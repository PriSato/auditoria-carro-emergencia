# Auditoria de Carros de Emergência

Aplicativo single-file (HTML/CSS/JS) para auditoria de carros de emergência
hospitalares, desenvolvido como Claude.ai Artifact.

## Arquivo principal
`Auditoria_Carro_de_Emergencia.html` — contém toda a interface, lógica e estilos.

## ⚠️ Importante
Este arquivo depende da API `window.storage`, disponível **apenas** dentro do
ambiente de artifacts do Claude.ai. Ele foi versionado aqui apenas como
backup/histórico de versões — para usá-lo, abra-o através do Claude.ai
(cole o conteúdo em um novo artifact ou continue a conversa onde ele foi criado).
Abrir o arquivo diretamente no navegador (file://) ou hospedá-lo fora do
Claude.ai fará as funções de salvar/excluir auditorias falharem, pois
`window.storage` não existirá nesse contexto.
